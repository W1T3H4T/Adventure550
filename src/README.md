# Source Files

This directory contains all C source files for Adventure550.

## Files

- **main.c** - Main game loop and initialization
- **init.c** - Game state initialization and save/load functionality
- **actions1.c** - Action handlers (part 1)
- **actions2.c** - Action handlers (part 2)
- **score.c** - Scoring system
- **misc.c** - Miscellaneous utility functions
- **datime.c** - Date/time handling (legacy FORTRAN compatibility)

All source files include headers from the `../include/` directory.
