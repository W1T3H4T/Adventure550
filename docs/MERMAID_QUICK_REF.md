# Mermaid Quick Reference

**Problem:** Visual Studio doesn't render Mermaid diagrams.

**Quick Solutions:**

1. **GitHub** ? Just push and view online (easiest)
2. **VS Code** ? Install + Markdown Preview Mermaid extension
3. **Online** ? Copy code to [mermaid.live](https://mermaid.live/)

- `maps/adventure-maps-mermaid.md`
